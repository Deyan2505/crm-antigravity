# CRM-Antigravity

Universal CRM system for small businesses.

## Setup Instructions

Since this is a new project, you need to install the dependencies once.

### 1. Install & Start Server (Backend)
Open a terminal, navigate to the `server` folder:
```bash
cd server
npm install
npm run dev
```
The server will run on http://localhost:3000

### 2. Install & Start Client (Frontend)
Open a **new** terminal, navigate to the `client` folder:
```bash
cd client
npm install
npm run dev
```
The client will run on http://localhost:5173

## Features
- Client Management
- Integration with external forms (Webhooks)
- Email Automation
